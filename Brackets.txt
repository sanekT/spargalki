Ctrl + E  -  ставим курсор на тег и открываются cssстили для этого тега

Ctrl + E  -  ставим курсор на цвете в файле стилей и откроется палитра, также если поставит курсор на функциях анимации, то можно настроить кривую бизье

Ctrl + K  -  ставим курсор на свойстве css и откроется всё об этом свойстве

Расширение Special HTML Character  -  при нажатии Alt + C откроется окно со спецсимволами

Расширение Brackets Color Palete  - пипетка для выбора цвета

Расширение CSS Fier  -  копируем выделенные теги и вставляем в файл стилей, расширение преобразует теги в селекторы к ним

Расширение QuickFormTool - позволяет вставлять элементы формы

Расширение Brackets Snippets - позволяет сохранять куски кода, при нажатии Alt + S откроется меню где мы можем по слову вставить кусок кода или новый код туда записать
